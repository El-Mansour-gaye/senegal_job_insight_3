# Accès aux données pour Power BI

Vous pouvez connecter Power BI directement à ce jeu de données en utilisant l'option "Obtenir des données > Web" et en saisissant l'URL suivante :

`https://votre-domaine-vercel.app/data/jobs_senegal_processed.csv`

Les données sont mises à jour chaque fois que le pipeline de scraping réussit et que les fichiers sont poussés sur le dépôt ou générés sur le serveur.
